Build with us: Support and Knowledge Base for Beginners — Day 2 Agenda

Build with us: Support & Knowledge Base for Beginners

Focus: Build software with AI assistance — use GitHub Copilot and VS Code to build an app that uses the agent from Day 1. Schedule: 9:00 AM – 3:00 PM

Time Duration Segment
9:00 – 9:20 20 min Tech Setup
9:20 – 9:40 20 min Day 2 Overview
9:40 – 10:00 20 min GitHub Copilot 101
10:00 – 12:00 120 min Guided Build: Connect the Agent (template app)
12:00 – 12:45 45 min Lunch
12:45 – 2:00 75 min Build Your Own
2:00 – 2:35 35 min Demos
2:35 – 3:00 25 min Wrap-Up & Next Steps

9:00 – 9:20 · Tech Setup

Get everyone's development environment ready before content begins.

  • Connect to Wi-Fi
  • Connect to Teams (for file sharing)
  • Open VS Code and confirm it launches
  • Confirm GitHub Copilot is signed in and working
  • Confirm the template app is downloaded and opens in VS Code

Facilitators circulate to clear any setup blockers early.

9:20 – 9:40 · Day 2 Overview

Set the context and the goal for the day.

  • What we're building: an app that uses the agent from Day 1
  • Show a finished result — what they will build today
  • How AI assists you throughout the build

9:40 – 10:00 · GitHub Copilot 101

A quick, practical intro so everyone can use Copilot effectively.

  • What Copilot does in VS Code (suggestions, chat, edits)
  • What a Markdown (.md) file is and why we use it (plain-text formatting for notes, docs, and prompts)
  • How to write a good prompt
  • Accept, reject, and iterate on suggestions
  • Quick hands-on: generate a small snippet together

10:00 – 12:00 · Guided Build: Connect the Agent (Template App)

Everyone builds the same app together using the provided template.

  • Tour the template app structure
  • Connect the Day 1 agent to the template app
  • Use Copilot to modify and extend the app step by step
  • Run the app and confirm it responds using the agent
  • Testing
  • Ask questions that should work
  • Confirm expected behavior

Facilitators walk the room and keep teams in sync.

12:00 – 12:45 · Lunch

12:45 – 2:00 · Build Your Own

Participants apply what they learned to their own use case.

  • Point the app at their own Day 1 agent
  • Use Copilot to customize the app for their scenario
  • Add or change features with AI assistance
  • Testing
  • Which questions should work?
  • Which questions should not work?

Facilitators coach teams one-on-one during this block.

2:00 – 2:35 · Demos

Companies optionally demo what they built:

  • Problem — what they set out to solve
  • Solution — the app they built
  • Lessons — what they learned building with AI assistance

2:35 – 3:00 · Wrap-Up & Next Steps

Close out the event and point participants toward what's next.

  • Recap what was built across the two days
  • Discuss options for future deployment
  • Share resources for continued learning
  • Q&A and closing
