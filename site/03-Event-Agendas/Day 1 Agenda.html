Build with us: Support and Knowledge Base for Beginners — Day 1 Agenda

Build with us: Support & Knowledge Base for Beginners

Focus: Build and evaluate a Technical Support / Knowledge Base agent in Azure AI Foundry. Schedule: 9:00 AM – 3:00 PM

Time Duration Segment
9:00 – 9:30 30 min Tech Setup
9:30 – 9:50 20 min Event Overview
9:50 – 10:20 30 min Introductions
10:20 – 12:00 100 min Create 1st Foundry Agent (guided)
12:00 – 12:45 45 min Lunch
12:45 – 2:00 75 min Create 2nd Agent (own files & prompt)
2:00 – 2:35 35 min Demos
2:35 – 3:00 25 min Prep for Day 2 & Wrap-Up

9:00 – 9:30 · Tech Setup

Get everyone connected and ready to build before content begins.

  • Connect to Wi-Fi
  • Connect to Teams (for file sharing)
  • Load Azure and Azure AI Foundry
  • Confirm each participant can sign in and open a Foundry project

Facilitators circulate to clear any access blockers early.

9:30 – 9:50 · Event Overview

Set the context and the goal for the day.

  • What we're building: Troubleshooting and Knowledge Base agents
  • Show a finished result — what they will build today
  • Quick tour of how the day flows

9:50 – 10:20 · Introductions

  • Each company shares their use case
  • Brief round-the-room so everyone learns from one another

10:20 – 12:00 · Create 1st Foundry Agent (Guided)

Everyone builds the same agent together using materials we provide.

  • We provide the prompt
  • Participants upload our sample files (Sample Files - Good)
  • Testing
  • Run predefined questions that should work
  • Run predefined questions that should not work
  • Deploy the agent
  • Best Practices: good data vs. bad data
  • Show what represents good data and bad data
  • Use two pre-made file sets (Sample Files - Good and Sample Files - Bad) — start with the good set, then switch to the bad set to show the difference in results

12:00 – 12:45 · Lunch

12:45 – 2:00 · Create 2nd Agent (Their Own)

Participants apply what they learned to their own use case.

  • Participants upload their own files
  • Participants write their own prompt
  • Testing
  • Which questions should work?
  • Which questions should not work?
  • Deploy the agent

Facilitators coach teams one-on-one during this block.

2:00 – 2:35 · Demos

Companies optionally demo what they built:

  • Problem — what they set out to solve
  • Solution — what they built
  • Lessons — what they learned

2:35 – 3:00 · Prep for Day 2 & Wrap-Up

Preview what's possible with code, and confirm everyone is ready for Day 2.

  • Show what can be done using coding (GitHub Copilot + VS Code)
  • Remind participants to have VS Code and GitHub Copilot installed
  • Remind participants to download the template app
  • Recap the day and answer questions
